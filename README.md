# 启物科技官方网站

## 📋 项目简介

启物科技官方网站 - 专注于下一代具身智能大脑，打造轮式人形机器人，为工业农业场景提供智能机器人解决方案。

## 🚀 快速开始

### 方法一：直接打开文件
```
file:///C:/Users/leo/Desktop/website/index.html
```

### 方法二：使用本地服务器
```bash
# 使用Python启动服务器
python -m http.server 8000

# 然后在浏览器访问
http://localhost:8000
```

### 方法三：使用预览脚本
```bash
# Windows
preview.bat

# Linux/Mac
./deploy.sh
```
