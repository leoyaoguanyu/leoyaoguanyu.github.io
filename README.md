# 启物科技官方网站

## 项目简介

启物科技官方网站是一个专注于智能机器人研发的现代化企业官网。网站展示了公司的核心产品"启物O1轮式人形机器人"，以及公司的技术实力、发展动态和联系方式。

## 技术栈

- **前端框架**: 原生HTML5 + CSS3 + JavaScript
- **样式设计**: 响应式设计，支持移动端和桌面端
- **图标库**: Font Awesome 6.0
- **字体**: Google Fonts (Inter字体家族)
- **邮件服务**: EmailJS集成
- **视频格式**: WebM + MP4支持

## 主要功能特性

### 🎨 视觉设计
- **现代化UI**: 简洁、科技感的设计风格
- **响应式布局**: 完美适配各种设备屏幕
- **紫色主题**: 统一的品牌色彩体系 (#8b5cf6, #a855f7)
- **动画效果**: 平滑的过渡和悬停效果

### 🧭 导航系统
- **下拉菜单**: 机器人产品展示
- **响应式导航**: 移动端汉堡菜单
- **面包屑导航**: 清晰的页面层级

### 📝 内容展示
- **产品介绍**: 详细的O1机器人信息
- **公司介绍**: 团队实力、技术优势、未来规划
- **新闻中心**: 公司动态和行业资讯
- **联系表单**: 完整的客户联系系统

### 🔧 交互功能
- **表单验证**: 完整的表单验证系统
- **验证码**: 图形验证码防机器人
- **邮件发送**: EmailJS集成，支持自动回复
- **视频播放**: 支持多种视频格式

## 开发指南

### 🛠️ 本地开发
```bash
# 克隆项目
git clone https://github.com/leoyaoguanyu/leoyaoguanyu.github.io.git

# 进入项目目录
cd leoyaoguanyu.github.io

# 直接在浏览器中打开HTML文件
# 或使用本地服务器
python -m http.server 8000
```
